# fastapi-server

## Build Setup

```bash
# create virtual env
# mac
$ python3 -m venv env

# win
$ py -3 -m venv env

# activate virtual environment
# mac
$ . env/bin/activate

# win
$ env\Scripts\activate

# install dependencies
$ pip3 install -r requirements.txt

# serve with hot reload at localhost:8002
$ uvicorn main:app --reload --port 8002

```
